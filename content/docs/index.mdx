## What is Shop Studio?

Shop Studio is a powerful multi-platform e-commerce management solution designed for cross-border sellers. Seamlessly manage your Amazon, Ozon, Shopee stores from one centralized dashboard.

## Key Features

### Multi-Platform Integration
- Connect all your e-commerce platforms (Amazon, Ozon, Shopee) in one place
- Unified dashboard for all your stores
- Support for major global marketplaces

### Centralized Operations
- **Order Management**: Process and fulfill orders from all platforms in one interface
- **Inventory Sync**: Real-time synchronization prevents overselling across channels
- **Product Listings**: Bulk upload and optimize listings across multiple platforms
- **Analytics**: Comprehensive insights across all sales channels

### Smart Automation
- Automated order processing workflows
- Inventory level monitoring and alerts
- Smart pricing and promotion tools
- Customizable automation rules

## Getting Started

### Prerequisites
- Node.js 18+ or Bun
- pnpm, npm, or yarn

### Quick Start

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Build for production
pnpm build

# Start production server
pnpm start
```

## Platform Connections

### Supported Platforms
- **Amazon**: Global marketplace integration
