## 什么是 Shop Studio?

Shop Studio 是强大的多平台电商管理解决方案，专为跨境卖家打造。从单一控制台集中管理亚马逊、Ozon、Shopee 店铺，数小时内高效运营你的跨境电商业务。

## 核心功能

### 多平台集成
- 一处连接所有电商平台（亚马逊、Ozon、Shopee）
- 统一管理所有店铺的仪表板
- 支持全球主流电商平台

### 集中化运营
- **订单管理**：在一个界面中处理和履行所有平台的订单
- **库存同步**：实时同步防止跨渠道超卖
- **商品上架**：批量上传和优化多个平台的商品列表
- **数据分析**：所有销售渠道的全面洞察

### 智能自动化
- 自动化订单处理工作流
- 库存水平监控和警报
- 智能定价和促销工具
- 可定制的自动化规则

## 快速开始

### 前置要求
- Node.js 18+ 或 Bun
- pnpm、npm 或 yarn

### 快速启动

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 启动生产服务器
pnpm start
```

## 平台连接

### 支持的平台
- **亚马逊**：全球市场平台集成
- **Ozon**：俄罗斯领先电商平台
- **Shopee**：东南亚市场平台
- 更多平台即将推出...

### 安全凭证管理
所有平台凭证均使用行业标准加密技术进行加密。您的 API 密钥和访问令牌安全存储，绝不共享。
